# SMART PRINT KIOSK SYSTEM

## Project Title

Smart Print / Xerox Kiosk System using Python and HTML

## Aim

To design and develop a simple Smart Print Kiosk system that allows a user to upload a file, select printing options, and automatically calculate the printing cost using Python.

## Objective

* To understand basic Python programming
* To learn how real-life billing systems work
* To create a simple user interface using HTML
* To perform automatic price calculation based on user choices
  
## Introduction

In colleges, libraries, and cyber cafés, students often need to take printouts. Normally, a person has to search for a printer shop and calculate the price manually based on the number of pages, print type, and paper quality.

This project solves that problem by creating a **Smart Print Kiosk System**. The system takes user input such as number of pages, print type, and paper type, and automatically calculates the total cost and displays the bill.

## Features

* Upload PDF / Document
* Enter number of pages
* Choose Black & White or Color print
* Select Normal or Thick paper
* Automatic price calculation
* Bill display system
* Simple beginner-friendly interface

## Technologies Used

* Python
* HTML
* CSS
* Basic Programming Concepts
* GitHub

## Working of the Project

1. The user opens the Smart Print Kiosk page
2. The user uploads a file (PDF/document)
3. The user enters the number of pages
4. The user selects print type (Black & White / Color)
5. The user selects paper type (Normal / Thick)
6. The Python program calculates the total cost
7. The system displays the final bill
8. Printing process starts

## Algorithm

1. Start the program
2. Display "SMART PRINT KIOSK"
3. Ask the user to upload a file
4. Ask the user to enter number of pages
5. Ask the user to choose print type
6. Ask the user to choose paper type
7. Set price per page based on print type
8. Add extra cost if paper is thick
9. Calculate total price
10. Display the bill
11. Show payment message
12. End the program
